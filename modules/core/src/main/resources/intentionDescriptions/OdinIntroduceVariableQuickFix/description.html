Detects repeated expressions that can be replaced with a variable.

This inspection identifies cases where introducing a variable can simplify code by avoiding redundant calculations. 


sum := calculate_total_price() + calculate_total_price()

Quick fix result:


total_price := calculate_total_price()
sum := total_price + total_price
